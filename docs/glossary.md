# Glossary

Short definitions for terms that appear across these notes.

## LLM

A large language model: an AI model trained to process and generate text, code, and other structured outputs from prompts and context.

## Tool

A specific capability an AI system can call to do something outside plain text generation, such as searching, reading a file, creating a calendar event, or running a calculation.

## Tool calling

A pattern where a model requests a structured tool action, receives the result, and uses that result to continue the task. See [Tool Calling](./using-ai/agents/tool-calling).

## Harness code

The code surrounding the LLM. Generally written in a general purpose programming language like TypeScript.
